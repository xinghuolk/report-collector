# Specs Index

Use this directory as a lifecycle-managed workspace:

- `active/` contains specs and roadmaps that still guide the next decision.
- `reference/` contains still-relevant roadmap, architecture, or scope context that should inform new specs but should not be executed directly.
- `archived/` contains completed, superseded, or historical specs with no current decision-making role.

Current active entry:

- `active/2026-04-22-financial-report-analysis-unified-roadmap.md`

Archive rule:

- Keep a spec active only when it is still on the roadmap, still describes work that is not done, and the code has not already implemented it.
- Move a spec to reference when its implementation phase is closed but it still contains useful future scope, constraints, field inventories, or architecture direction.
- Archive completed specs, superseded specs, and specs whose remaining items are no longer on the current roadmap or are already summarized elsewhere.
