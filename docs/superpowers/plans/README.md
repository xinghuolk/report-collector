# Plans Index

Use this directory as a lifecycle-managed workspace:

- `active/` contains implementation plans that should still be executed.
- `reference/` contains plans kept only as implementation pattern references, if any.
- `archived/` contains completed, superseded, or historical implementation plans.

Current active implementation plans:

- None.

Archive rule:

- Keep a plan active only when it is still on the roadmap, contains unfinished implementation work, and the code has not already completed that work.
- Move a plan to reference only when its implementation pattern is likely to be reused but the exact task sequence is no longer executable.
- Archive completed plans, superseded plans, and unchecked plans that have been replaced by a later implemented plan.
